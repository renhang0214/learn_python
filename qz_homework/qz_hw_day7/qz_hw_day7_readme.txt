# 登录系统：需要实现注册，登录，删除，

管理员账号：admin
密码：123456

分别写出验证用户名，验证用户输入次数，验证用户是否被锁定，注册，登录，删除，main（主函数）
运行系统函数main，开始死循环（用户输入注册、登录、删除、或者退出），根据用户输入的编号，分别执行函数
编号1：注册
    先执行验证用户名函数（test）并判断返回值，true表示用户存在，返回重新循环，false表示用户不存在，执行（register）注册成功
编号2：登录
    先执行验证用户名函数（test）并判断返回值，true表示用户存在，
        ==》》 执行验证用户是否被锁定函数（test_num）并判断用户是否被锁定，true表示用不未被锁定
            ==》》 执行验证用户输入次数函数（user_num）并判断用户是否被锁定，true表示用户未被锁定
                ==》》 执行登录函数（login）
            ==》》 false表示用户输入次数超限，锁定用户
        ==》》 false表示用户被锁定，返回重新循环
    false表示用户不存在返回重新循环，
编号3：删除
    先输入管理员账户及密码，并进行验证，不正确返回重新开始循环，正确：输入准备删除的用户，执行test函数，验证用户是否存在，
    存在：删除成功，不存在返回重新开始循环
